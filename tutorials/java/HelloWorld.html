Hello World!

by Jake Barnes

Posted on November 2, 2017 at 12:04 PM

This article is going to show you how to write your very first Java program. We are simply going to output the phrase “Hello World!” to the console line.

Firstly you'll need to open up your favourite IDE (I'll be using Intellij for these tutorials) and create a new project. Then in your java class you'll need to type out:

public static void main(String args[]) {
System.out.println(“Hello World!”);
}

Now, lets quickly break down what each line does.

public static void main(String args[])
This line will be present in every single Java program you write. It is the main method of the application indicated by the word “main” and there can only be one main method in an application. It indicates where the program should start. there are also some other words in this line such as public, static, void and the words in the parenthesis, but for now we aren’t going to worry about this as it isn’t essential for you to understand right now.

System.out.println(“Hello World!”);
This line uses a method called println() to print content to the command line. We have passed in something called a string with the value “Hello World!”;
